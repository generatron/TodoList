## Installing and Running

1. Install
	
	`npm install -g cordova ionic`
	
2. Get dependencies:

	`npm install`
	
	`bower install`
	
3. Run It 

   `ionic serve`

   or
   
   `ionic platform add ios`
   
   `ionic build ios`
   
   `ionic emulate ios`
